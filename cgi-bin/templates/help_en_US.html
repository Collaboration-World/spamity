[% i18n('appendix: filter types') %]

  • rbl - Message rejected by a Realtime Blackhole List
  • rhsbl client - Message rejected by Right Hand Side Block List
  • header date - Message has a date from the past or the future
  • header suject - Message rejected by suspicious subject; filter method starts with the letter h
  • header x-mailer - Message rejected by suspicious mailer
  • header content-disposition - Message rejected by suspicious attachment
  • header content-type - Message rejected by suspicious attached file; filter method specifies the file extension
  • body - Message rejected by suspicious body content; filter method starts with the letter b
  • access username - Message rejected by access username
  • virus - Message rejected by the antivirus software; filter method starts with virus-
  • spam - Message rejected by SpamAssassin; filter method starts with spam-
  • banned - Message rejected by prohibited attached file
  • helo - Message rejected by an unacceptable or missing HELO command.
